# Menu icons — drop-in slots

The game references the PNGs below. Any that are **missing are silently skipped**
(the `<img onerror>` removes itself), so the menus stay clean text until you add them.
Generate them in the same neon style as the loading screens.

**Style:** transparent background PNG, square, simple **white / light neon line-art**
(they sit on dark UI and are tinted by the row highlight). ~96×96 px is plenty.

## Home menu (left of each button)
| file | icon |
|------|------|
| `icon_continue.png` | play triangle ▶ |
| `icon_newgame.png`  | plus / new + |
| `icon_online.png`   | globe |
| `icon_garage.png`   | car (front) |
| `icon_missions.png` | star or flag |
| `icon_store.png`    | shopping cart |
| `icon_settings.png` | gear |
| `icon_exit.png`     | door / logout arrow |

## Player card (top-right of the home screen)
| file | icon | size |
|------|------|------|
